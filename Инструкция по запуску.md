# Инструкция по запуску автоматизированных тестов API микросервиса объявлений

## 1. Требования

Для запуска тестов необходимо наличие установленного **Python 3.6+** и следующих библиотек:

*   `pytest`
*   `requests`

## 2. Установка зависимостей

1.  Перейдите в директорию проекта:
    ```bash
    cd ads_api_tests
    ```

2.  Установите необходимые зависимости, используя файл `requirements.txt`:
    ```bash
    pip install -r requirements.txt
    ```
 
## 3. Запуск тестов

Запуск всех тестов осуществляется с помощью команды `pytest` в корневой директории проекта (`ads_api_tests`):

```bash
pytest test_ads_api.py
```
